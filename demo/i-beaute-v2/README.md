# i-Beaute · Sofwave — v2 (파이프라인 생성본)

같은 i-Beaute 사실을 **에이전트 체인(intake → research → copy → design → assemble)** 으로 새로 만든 A/B 변형입니다.
핸드크래프트(v1)가 *결과중심*("Lift. Without the surgery.")이라면, v2는 리서치 에이전트가 잡은
**"정직한 상담"을 척추로 한 신뢰중심** 앵글입니다 — "We'll tell you when Sofwave isn't the answer."

- **라이브:** https://i-beaute-sofwave-v2.pages.dev
- **브리프(입력):** [`briefs/i-beaute-v2.json`](../../briefs/i-beaute-v2.json)
- **파이프라인 중간 산출물:** [`pipeline/`](pipeline/) — 각 에이전트 단계의 실제 출력
  - `intake.json` (01) → `research.json` (02) → `copy.json` (03) → `theme.json` (04) → `briefs/i-beaute-v2.json` (05 조립) → `index.html`

> 같은 사실 → 전혀 다른 설득 전략이 나오는 것을 보여주는, 프레임워크 흐름의 실제 예시.
> 정직성 유지: 실제 Fresha 4.9★/193 + 실제 클리닉 후기, 상담 기반 가격(가짜 수치·후기·긴급성 없음).
